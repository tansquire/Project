IIT MADRAS water network control and monitoring system
Real time redundant data acquisition system using LPWAN and GSM network
Copyright@Process Control Lab, IIT Madras
Redandant communication system with LoRA LPWAN and GSM. In case of LoRA failure, link will switch over to GSM. The system has been designed with very low cost and reliable sensor, actuator, and controller.The sensors and actuetors are connected to Arduino which is further connected to Raspbery Pi through Lora. Raspbery Pi is connected to IITM Network using ethernet. For secondary communication, each Arduino is using a GSM modem with a SIM, and using subscribing/publishing technique (MQTT) to comunicate to a cloud server.
